stat-nlp-course
===============

Code for the UCL Statistical NLP course

```
  cd workspace
  git clone https://github.com/wolfe-pack/wolfe.git
  git clone https://github.com/uclmr/stat-nlp-course.git
  cd stat-nlp-course
  sbt compile
```

```
sbt gen-idea
```
